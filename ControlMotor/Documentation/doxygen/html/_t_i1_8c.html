Detailed Description

This component "TimerInt" implements a periodic interrupt. When the component and its events are enabled, the "OnInterrupt" event is called periodically with the period that you specify. TimerInt supports also changing the period in runtime. The source of periodic interrupt can be timer compare or reload register or timer-overflow interrupt (of free running counter).
